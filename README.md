# SimpleScheduler
## Architecture
- Web Server: Nginx
- AP Server: uWsgi / ASGI
- DB Server: PostgreSQL

**In the future, I will deploy this application to AWS ECS, RDS and S3.**

## SetUp
### Git Hooks
```
make git-hooks
```
